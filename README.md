# 10X Studio

Plataforma de creación, gestión y publicación de contenido con IA.

## Setup

```bash
npm install
cp .env.local.example .env.local
# Agrega tu ANTHROPIC_API_KEY en .env.local
npm run dev
```

## Extensión Chrome

Instala `substack-extension/` en `chrome://extensions` (modo desarrollador).

## Stack

- Next.js 14 · TypeScript · Tailwind CSS · Inter font
